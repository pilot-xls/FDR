Algorithm notes

Airport names are attempted online from coordinates. If that fails, the app asks for a manual sector name.

  • Flight time: from airborne/initial climb detection to landing detection.
  • Block time: from blocks off to blocks on.
  • Next sector: closes the current sector, saves it, and immediately starts the next one.
  • Offline: GPS recording works offline, but online airport lookup does not.
